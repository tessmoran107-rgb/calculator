# Calculator
A simple, functional calculator web application built with HTML, CSS, and JavaScript as part of The Odin Project curriculum.

## Description
This calculator performs basic arithmetic operations including addition, subtraction, multiplication, and division. It features a clean interface and handles sequential calculations, allowing users to chain operations together.

## Features
Basic arithmetic operations (+, -, ×, ÷)
Clear button to reset calculations
Chained calculations (result of one operation can be used in the next)
Automatic clearing when starting a new calculation after pressing equals
Responsive button interface

bashcd calculator

Open index.html in your web browser

## Usage
Click number buttons to input values
Click an operator button (+, -, ×, ÷) to select an operation
Click another number to complete the expression
Press the equals button (=) to see the result
Use the clear button (C) to reset the calculator
